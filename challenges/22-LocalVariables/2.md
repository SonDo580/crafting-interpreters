# Challenge 22.2

How do other languages handle code like this:

```
var a = a;
```

What would you do if it was your language? Why?

## Answer

- **JavaScript**: depends on whether we use `let` or `var` keyword

```js
let a = "outer";
{
  let a = a; // ReferenceError: Cannot access 'a' before initialization
}
// -> same behavior as Lox


var a = "outer";
{
    var a = a; // a is now "outer"
}
func x() {
    var a = "outer";
    {
        var a = a; // a is now "outer"
    }
}

// Reason: variables defined with 'var' have function scope or global scope (not block scope) and is hosted with the default value of undefined
```
